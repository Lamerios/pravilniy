# 🌱 Database Seeders

Этот каталог содержит сиды с тестовыми данными для разработки Quiz Game.

## 📂 Структура

```
seeders/
├── README.md                 # Этот файл
├── index.ts                  # Основной экспортер сидов
├── basic-seeder.ts          # Базовые сиды (организации, пользователи)
└── demo-game-seeder.ts      # Демо игра с полным циклом данных
```

## 🎯 Назначение сидов

### 1. **basic-seeder.ts** - Основные данные
- 🏢 Demo Organization (enterprise план)
- 👤 Admin и User аккаунты
- 📋 Базовые шаблоны игр
- 👥 Справочник команд (20 команд)

### 2. **demo-game-seeder.ts** - Полная демо-игра  
- 🎮 Завершенная игра "Demo Quiz Night"
- 🏆 8 команд-участников
- 📊 6 раундов с реальными результатами
- 🎯 Система баллов и ставок

## 🚀 Использование

```bash
# Запуск всех сидов
npm run db:seed

# Сброс и пересоздание с сидами
npm run db:reset

# Только базовые данные
npm run seed:basic

# Только демо игра  
npm run seed:demo
```

## 📊 Тестовые данные

### Организация
- **Название**: Demo Organization  
- **Домен**: demo.quiz-game.com
- **План**: Enterprise (без ограничений)

### Пользователи
```
admin@demo.quiz-game.com / Admin123!
user@demo.quiz-game.com  / User123!
```

### Команды
20 команд с реалистичными названиями:
- "Мозговой штурм", "Эрудиты", "Знатоки" и др.
- У каждой команды есть логотип и описание

### Демо игра
- **Дата**: Последняя пятница  
- **Статус**: Завершена
- **Участники**: 8 команд
- **Раунды**: 6 тематических раундов
- **Результаты**: Реалистичные баллы и позиции

## 🔄 Обновление данных

При изменении схемы БД обновите соответствующие сиды:

1. Отредактируйте нужный seeder файл
2. Запустите `npm run db:reset` для пересоздания
3. Проверьте данные в PgAdmin или через API
